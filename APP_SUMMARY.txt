CHAT SYSTEM FEATURES (Your Spec)
───────────────────────────────────────────────────────────────────────────────

✅ Chat Types
   DM → 1:1 private conversations
   GROUP → Ministry/team chats
   SMALL_GROUP → Community chats
   SERVICE → Time-bound operational chats

✅ Inbox Layout
   Section 1: Direct Messages (DM badge)
   Section 2: Groups (GROUP/SMALL_GROUP badges)
   Section 3: Upcoming Services (SERVICE badge)

✅ Notification Logic
   DM: Always HIGH priority
   GROUP: NORMAL, user can mute
   SERVICE: HIGH priority (time-sensitive)
   Quiet Hours: Customizable (default 22:00-08:00)
   Muting: Individual, category, or global

✅ Service Chat Flow
   1. Service instance created → Chat auto-created
   2. Volunteer assigned → Auto-added to chat
   3. Chat shows service info header
      - Call time
      - Location
      - Setlist link
   4. Volunteer can see team
   5. Leader can communicate

✅ Access Control (RLS)
   DM: Only 2 users can see
   GROUP: Only group members can see
   SMALL_GROUP: Only group members can see
   SERVICE: Only assigned volunteers can see
