QUICK DEPLOY GUIDE
═══════════════════════════════════════════════════════════════════════════════

1. CREATE SUPABASE STORAGE BUCKET (5 minutes)
   ├─ Go to: https://app.supabase.com
   ├─ Select your project
   ├─ Go to: Storage (left sidebar)
   ├─ Click: "Create a new bucket"
   ├─ Name: "media"
   ├─ Toggle: Make it Public
   └─ Click: Create bucket

2. INSTALL DEPENDENCIES
   npm install

3. RUN LOCALLY
   npm run dev
   # Scan QR with Expo Go app on your phone

4. TEST MEDIA UPLOAD
   ├─ Go to Social tab
   ├─ Create a post
   ├─ Tap "Add Media" or use MediaPicker
   ├─ Take a photo or choose one
   ├─ Should upload and show preview
   └─ Complete post and save

5. DEPLOY TO PRODUCTION
   # Web: npm run build:web → deploy to Vercel/Netlify
   # Mobile: Use EAS Build or export APK/IPA
